# Prototipo de un videojuego roguelike con mecánicas de combate táctico y construcción de mazos

Repositorio del proyecto de Trabajo de Fin de Máster (TFM) de **Carlos Castellano Jacob** para el Máster en Diseño y Programación de Videojuegos de la UOC.

## 1. Descripción del Proyecto

Este proyecto consiste en el desarrollo de un prototipo jugable en Unity que fusiona tres pilares de diseño:

* **Deckbuilding:** Cada protagonista de un grupo posee su propio mazo de cartas personalizable.
* **Combate Táctico por Turnos:** Un sistema de combate estratégico donde se gestionan las acciones de cada personaje.
* **Estructura Roguelike:** Progresión a través de un mapa de nodos ramificados, con eventos aleatorios, alta rejugabilidad y gestión de recursos en un campamento.

El objetivo principal es entregar un prototipo pulido que demuestre la viabilidad y sinergia de estos sistemas en un *core loop* de juego completo.

## 2. Entorno Técnico

* **Motor:** Unity 2022.3.x (LTS)
* **Lenguaje:** C# 
* **IDE:** JetBrains Rider
* **Control de Versiones:** Git (alojado en GitLab)
* **Gestión de Datos:** Los datos de cartas, enemigos y eventos se gestionarán externamente (p.ej., JSON) para facilitar la iteración.
